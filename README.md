# Customs Clearance App - Flutter + SQLite

هذه نسخة بداية كاملة لتطبيق التخليص الجمركي.

## ماذا يوجد في التطبيق؟

- شاشة رئيسية.
- شاشة العامل لإرسال بيانات:
  - اسم الوكيل
  - اسم السائق
  - رقم اللوحة
  - الكمية
- شاشة المستخدم المستلم:
  - تظهر أسماء الوكلاء فقط.
  - عند الضغط على الوكيل تظهر التفاصيل.
  - قبول أو رفض.
- عند القبول:
  - يتم إنشاء سجل في جدول التخليص تلقائياً.
- شاشة جدول التخليص:
  - اسم الوكيل
  - اسم السائق
  - رقم اللوحة
  - الكمية
  - الوحدة
  - سعر الوحدة
  - مبلغ الجمارك
  - التاجر المستفيد
- عند الضغط على عمود التاجر المستفيد:
  - تظهر نافذة إضافة اسم التاجر.
- عند الضغط على زر التسعير:
  - تظهر الكمية تلقائياً.
  - يضاف سعر الوحدة.
  - يتم اختيار الوحدة: طن / كيس / كيلو.
  - يحسب النظام مبلغ الجمارك = الكمية × سعر الوحدة.

## طريقة التشغيل

1. أنشئ مشروع Flutter جديد:

```bash
flutter create customs_clearance_app
```

2. انسخ مجلد `lib` وملف `pubspec.yaml` من هذه الحزمة إلى المشروع.

3. نفذ:

```bash
flutter pub get
```

4. شغل التطبيق:

```bash
flutter run
```
